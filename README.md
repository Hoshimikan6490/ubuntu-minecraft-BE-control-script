# ubuntu-minecraft-BE-control-script
ubuntuで、マイクラ統合版サーバーを管理するためのスクリプト集です。

## 概要
基本的に、スクリプト及びディレクトリは以下のようになっています。
```
統合版マインクラフトの親フォルダ
　┣━「serverUpdateTemp」：　サーバーをアップデートする際に残しておく必要があるデータを一時的に避難させるための一時フォルダです。
　┣━「server」：　統合版のサーバーデータは全てここに保管されます。（サーバーデータの表記は省略します。）
　┣━「backups」：　バックアップスクリプトが生成したバックアップがここに保管されます。
　┃　┣━「monthly」：　毎月１日に生成される月間バックアップが最大３世代までzipファイルで保管されます。
　┃　┗━「daily」：　毎日３回生成される毎日のバックアップが最大３世代までzipファイルで保管されます。
　┣━「backup.sh」：
　┣━「log.txt」：
　┣━「reboot.sh」：
　┣━「run.sh」：
　┣━「stop.sh」：
　┣━「update.sh」：
　┗━「version.txt」：　
```
