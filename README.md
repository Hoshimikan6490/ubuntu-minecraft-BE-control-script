# ubuntu-minecraft-BE-control-script
ubuntuで、マイクラ統合版サーバーを管理するためのスクリプト集です。

## 概要
基本的に、スクリプト及びディレクトリは以下のようになっています。
```
統合版マインクラフトの親フォルダ
　┣━「serverUpdateTemp」：　サーバーをアップデートする際に残しておく必要があるデータを一時的に避難させるための一時フォルダです。
　┣━「server」：　統合版のサーバーデータは全てここに保管されます。（サーバーデータの表記は省略しますが、このフォルダ内に「server.properties」などのファイルが保管されます。）
　┣━「backups」：　バックアップスクリプトが生成したバックアップがここに保管されます。
　┃　┣━「monthly」：　毎月１日に生成される月間バックアップが最大３世代(※1)までzipファイルで保管されます。
　┃　┗━「daily」：　毎日３回生成される毎日のバックアップが最大３世代(※1)までzipファイルで保管されます。
　┣━「backup.sh」：　サーバーファイルを毎日/月間でバックアップするスクリプトです。
　┣━「log.txt」：　各スクリプトによる操作のログが記録されます。
　┣━「reboot.sh」：　統合版のサーバーを再起動するスクリプトです。
　┣━「run.sh」：　統合版のサーバーを起動するスクリプトです。
　┣━「stop.sh」：　統合版のサーバーを停止するスクリプトです。
　┗━「update.sh」：　統合版のサーバーを更新するスクリプトです。
```
※1: 「common.sh」の「NUM_GENS」パラメータから変更できます。

## 設定項目
全てのshスクリプトは、common.shの設定を基準に動作します。必ず手動設定部分を設定したうえで実行してください。
